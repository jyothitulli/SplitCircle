# SplitCircle — Phase 9 + 10

## Backend (Phase 9)
- Phase 9A: OCR Receipt Scanning (Tesseract.js + Cloudinary)
- Phase 9B: Voice Expense Logging
- Phase 9C: AI Insights (Gemini)
- Phase 9D: Conflict Prediction

### Run Backend
```bash
cd backend
npm install
npm run dev
```

## Frontend (Phase 10)
Full React SaaS dashboard.

### Tech Stack
- React 18 + Vite
- Tailwind CSS (dark theme)
- React Router DOM v6
- TanStack React Query v5
- Axios
- React Hook Form
- Recharts
- Framer Motion
- React Hot Toast

### Pages
- Login / Register
- Dashboard
- Circles
- Expenses
- Balances
- Settlements
- Chores
- Fairness
- OCR Upload
- AI Insights

### Run Frontend
```bash
cd frontend
npm install
npm run dev
```
Open http://localhost:5173

### Environment Variables
```env
VITE_API_URL=http://localhost:4000
```
